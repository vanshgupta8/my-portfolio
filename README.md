# Vansh Gupta Resume Portfolio

An interactive resume-style portfolio for Vansh Gupta, a Computer Science Engineering student specializing in AIML.

## Live Demo

[https://https-portfolio-website-one-kappa-2.vercel.app](https://https-portfolio-website-one-kappa-2.vercel.app)

## Sections

- Professional summary
- Education
- Technical skills
- Resume projects
- Coding achievements
- Contact details

## Features

- Responsive layout for desktop and mobile
- Dark and light mode toggle
- Animated counters
- Project filters and project detail modal
- Contact form connected to Gmail through FormSubmit
- Links for LinkedIn, LeetCode, CodeChef, GitHub, and resume

## Tech Stack

- HTML
- CSS
- JavaScript
- Vercel deployment
